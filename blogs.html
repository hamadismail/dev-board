Question-1: What are the different ways to select an element in the DOM?

Selecting by Tag Name (getElementsByTagName),
Selecting by ID (getElementById),
Selecting by Class Name (getElementsByClassName),
Selecting with CSS Selectors (querySelector and querySelectorAll).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML - Gets or sets HTML content inside an element.
innerText - Gets or sets only visible text, ignoring hidden elements.
textContent - Gets or sets all text, including hidden elements.

Question-3: What is event delegation in the DOM?

Is a technique where a single parent element handles events for multiple child elements.

Question-4: What is event bubbling in the DOM?

Is a process where an event starts from the target element and propagates up through its parent elements.

Question-5: How do you create, add, and remove elements using JavaScript?

1 -> Creating an Element (document.createElement)
2 -> Adding an Element (appendChild)
3 -> Removing an Element (removeChild)

// creating element
-------------------
const li = document.createElement("li")
li.innerText = "JavaScript id fun!"

// adding element to the target element
-----------------------------------
const ul = document.getElementByID('target-id');
ul.appendChild(li);

// remove the element
----------------------
ul.removeChild(li);
